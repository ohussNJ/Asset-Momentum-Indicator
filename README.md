# Asset Momentum Indicator
* [TradingView](https://www.tradingview.com) script to track momentum of an asset
* It is best used for tracking divergences and finding local tops/bottoms
* To setup go to Pine Editor -> Select 'Create new' -> Select 'Indicator' -> Add script into editor -> Save script -> Go to desired chart -> Go to 'Indicators' -> Select 'My scripts' -> Enable the script
